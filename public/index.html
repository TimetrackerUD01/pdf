รูปภาพเป็น PDF

แปลงไฟล์รูปภาพ (.jpg, .png, .gif) เป็น PDF คุณภาพสูง

Text เป็น PDF

แปลงไฟล์ข้อความ (.txt) เป็น PDF พร้อมการจัดรูปแบบ

ลดขนาด PDF

ลดขนาดไฟล์ PDF โดยไม่สูญเสียคุณภาพ ทำให้ส่งและเก็บง่ายขึ้น

รวม PDF

แยก PDF

แยกไฟล์ PDF เป็นหลายไฟล์ตามหน้าที่ต้องการ หรือแยกทีละหน้า
